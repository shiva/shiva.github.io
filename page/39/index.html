Jan 31, 2007

Ideas: Plugins for Windows Live writer
Jan 30, 2007

Been using Windows Live Writer as my blog writing tool. Here are some plugins that I would definetly make my life much easier. If you are insterested in writing a plugin or would like to add to these requirements, do get in touch. Extended Link insertion Link to different Search Engines (Google, Google Blog Search, Code search, Sun API search, Technocrati search etc.) Link to various product sites (Amazon, RitzCamera etc) Link to previously exisitng Posts Recent Links Links from clipboard Auto-link insertion based on certain keywords Multiple link inclusion - when this mode is enabled every word that is clicked on / selected should be added to a bucket, which can then be used to create links on.
